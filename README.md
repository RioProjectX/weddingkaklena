## 🚀 Installation

Clone repository:

```bash
git clone <repository-url>
```

Masuk ke folder project:

```bash
cd project-name
```

Install dependency:

```bash
npm install
```

Menjalankan development server:

```bash
npm run dev
```

Build production:

```bash
npm run build
```
